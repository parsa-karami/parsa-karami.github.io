## Projects
- **Machine Learning-Based Speed Detection System Using ADXL345 and GPS Validation**  
In this project, I developed a system to estimate vehicle speed from acceleration data collected using the ADXL345 sensor. To validate the results, speed measurements from GPS coordinates recorded by the NEO-M8 module were used. Key steps included feature engineering, such as computing acceleration differentiation, to enhance model performance. I implemented machine learning algorithms, including Linear Regression and Decision Tree Regression, and evaluated a Neural Network to compare performance in predicting speed.  
[View Code](https://github.com/parsa-karami/Machine-Learning-Based-Speed-Detection-System-with-ADXL345-and-GPS-Validation.git)
- **Driving Cycle**  
I led my team in the Rah Neshan Competition, where we analyzed Tehran's driving cycles to determine optimal locations for electric bus charging stations, the primary goal of the event. To achieve this, we defined micro-trips and clustered data into categories based on route types and traffic conditions. Our innovative approach earned us fourth place in the competition.

- **Smart Bag**  
My friend and I developed an innovative smart bag to help users avoid forgetting essential items, such as school books. The system employed RFID technology, using RFID tags to detect and track items placed in the bag and triggering an alarm when an item was missing. This solution's adaptability makes it suitable for various applications across diverse settings.
