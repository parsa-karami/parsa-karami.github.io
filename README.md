## Projects
- **Machine Learning-Based Speed Detection System Using ADXL345 and GPS Validation**  
In this project, I developed a system to estimate vehicle speed from acceleration data collected using the ADXL345 sensor. To validate the results, speed measurements from GPS coordinates recorded by the NEO-M8 module were used. Key steps included feature engineering, such as computing acceleration differentiation, to enhance model performance. I implemented machine learning algorithms, including Linear Regression and Decision Tree Regression, and evaluated a Neural Network to compare performance in predicting speed.
